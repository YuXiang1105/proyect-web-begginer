Before using, follow the next steps:
1-Install the virtual enviroment with the next commands:
python3 -m venv .venv
source .venv/bin/activate (mac and linux)
source .venv/Scripts/activate (Windows)
2-Install the next packages in the virt.environment:
pip install -r requirements.txt
3-For use,write the next command in the powershell:
python run.py
4-Open app at http://127.0.0.1:5000
