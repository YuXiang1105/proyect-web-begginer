# Relic Flask Boilerplate

This is a basic Flask project scaffold using the application factory pattern.

## Setup Instructions

1. Create and activate a virtual environment:
   ```bash
   python3 -m venv venv
   source venv/bin/activate
   ```

2. Install dependencies:
   ```bash
   pip install -r requirements.txt
   ```

3. Run the app:
   ```bash
   python run.py
   ```

You should see your Dev Temple welcome screen at `http://localhost:5000`.
